ODBC와 ASP 환경 설정

ODBC 요구사항 처리를 위한 클라이언트 환경을 설정하려면 CUBRID ODBC Driver가 설치되어 있어야 한다. CUBRID ODBC Driver는 CUBRID 설치 시 자동으로 설치된다. 서버나 클라이언트 어느 것을 설치해도 된다. 설치된 CUBRID ODBC Driver를 확인하려면 [제어판]의 [관리 도구]에서 [데이터 원본(ODBC)]을 실행한다.

 

CUBRID ODBC Driver가 확인되었다면 응용 프로그램에서 접속하려는 데이터베이스로 DSN을 설정한다. DSN 설정을 위해 ODBC 데이터 원본 관리자에서 [추가] 버튼을 클릭하면 다음과 같은 대화 상자가 나타난다. 'CUBRID Driver'를 선택하고 [마침] 버튼을 클릭한다.

 

다음과 같이 [Config CUBRID Data Source] 팝업 창이 나타나면 [DB Name]에는 접속하려는 데이터베이스 이름을, [Server Port]에는 CUBRID 브로커의 포트 번호를 입력하고 [OK] 버튼을 클릭한다.
