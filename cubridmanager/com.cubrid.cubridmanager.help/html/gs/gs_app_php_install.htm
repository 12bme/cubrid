PHP 모듈 설치

CUBRID 홈페이지(http://www.cubrid.com)에서 PHP 모듈을 다운로드한다.

Windows 버전 설치

PHP가 설치되어 있는 디렉터리에 CUBRID라는 디렉터리를 만든 후 다운로드한 cubrid_err.msg 파일과 cubrid_php 버전 .dll 파일을 복사한다.
다운로드한 CUBRID PHP 모듈 버전이 5.1.4라면, 다음 예제와 같이 php.ini를 편집하여 관련 설정 값을 추가한다.

extension_dir=C:\PHP\CUBRID
extension=cubrid_php5.1.4.dll
[CUBRID]
cubrid.err_path=C:\PHP\CUBRID

설정이 완료되면 웹 서버를 재시작한다. PHP의 phpinfo() 함수를 사용하여 PHP 설정 내용을 확인했을 때 다음과 같이 CUBRID가 보이면 정상적으로 설치가 완료된 것이다.

Linux 버전 설치

Linux 버전은 Windows 버전과는 달리 소스코드가 제공된다. 사용자는 CUBRID 홈페이지(http://www.cubrid.com)에서 소스코드를 다운로드한 다음 압축 파일에 포함되어 있는 INSTALL 파일 내의 설치 가이드에 따라 설치를 진행한다. Linux 버전도 다음과 같이 php.ini 파일을 편집하고 저장한 후 웹 서버를 재시작한다.

; Directory in which the loadable extensions (modules) reside.
extension_dir = /usr/lib/php5/lib/php/ext
extension=cubrid.so
;Windows의 경우 extension=cubrid.dll
[CUBRID]
cubrid.err_path=/home/cubrid/CUBRID/msg

phpinfo() 함수를 사용해서 확인한다.

[root@localhost]#vi test.php

웹 브라우저를 통해서 확인했을 때 Windows 버전과 마찬가지로 CUBRID가 보인다면 정상적으로 설치가 완료된 것이다.
