브로커 메뉴

탐색 트리에서 [브로커]를 마우스 오른쪽 버튼 클릭하면, 브로커 전체에 대한 기능을 수행할 수 있다. [상태 보기], [브로커 시작], [브로커 정지], [속성], [새로고침]으로 구성되어 있다.

브로커가 시작되어 있으면 탐색 트리에서 아이콘으로 표시되고 정지 상태이면 아이콘으로 표시된다.

상태 보기

[상태 보기]는 다음 명령어와 같은 기능을 수행한다.

cubrid broker status

자동으로 새로운 정보를 보려면, [속성]의 [새로고침 설정]에서 [켜기]를 선택하고 새로 고침 간격을 설정한다.

브로커 시작/정지

브로커 전체를 시작/정지하는 기능으로 $CUBRID/conf/cubrid_broker.conf 파일의 브로커 구성 정보를 기반으로 시작 및 정지를 수행한다.

속성

브로커의 속성을 편집하려면 탐색 트리에서 호스트를 마우스 오른쪽 버튼 클릭하여 [속성]을 선택하고 [브로커 변수 설정]을 클릭하거나, [브로커]를 마우스 오른쪽 버튼 클릭하고 [속성]을 선택한다.

$CUBRID/conf/cubrid_broker.conf 파일내의 모든 구성 정보를 GUI로 추가, 편집, 삭제할 수 있다. 각 매개 변수에 대한 자세한 설명은 브로커 설정을 참조한다.

브로커에서 설정하는 모든 값은 입력 또는 편집 시 바로 유효성 검사를 수행하여 사용자의 실수를 최소화한다.
