Eclipse 플러그인

Eclipse 플러그인으로 CUBRID 매니저 클라이언트에 접속하면, [CUBRID] 메뉴에서 기본 메뉴를 제공하고 트리 창의 기능은 RCP 응용 프로그램과 동일하다. CUBRID 매니저 플러그인을 실행하려면, Eclipse에서 [Window] > [Open Perspective] > [Other]를 선택한 후, [CUBRID Manager]를 선택한다.

 

 

CUBRID 메뉴

[CUBRID] 메뉴는 구동 및 설정 그리고 추가에 대한 기본 메뉴만 제공한다. 다른 기능은 RCP 응용 프로그램과 같이 트리 창에서 마우스 오른쪽 버튼을 클릭하여 실행할 수 있다.

 

툴바

CUBRID 매니저 Eclipse 플러그인의 툴바는 호스트 추가, 데이터베이스 생성, 질의 편집기, 그리고 서비스 시작/정지 기능을 제공한다.
