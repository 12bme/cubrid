DATETIME

설명

DATETIME 타입은 연도, 월, 날짜, 시, 분, 초, 밀리초 로 이루어진 값이다. DATETIME 리터럴은 문자 스트링으로 표현하는데, 키워드 DATETIME 뒤에 다음과 같은 형식으로 작성한다

datetime ‘mm/dd[/yyyy] hh:mm[:ss[.ff]] [am | pm]’
datetime ‘[yyyy-]mm-dd hh:mm[:ss[.ff]] [am | pm]’

mm은 월, dd는 날짜, yyyy는 연도, hh는 시간, mm은 분, ss는 초, ff는 밀리초를 나타낸다. 연도 부분을 생략하면 현재 연도가 자동으로 지정된다. 시간, 분, 초를 생략하면 12:00:00.000로, 밀리초 부분을 생략하면 000밀리초로 자동으로 지정된다

참고 사항

모든 항목은 정수 형태로 입력되어야 한다.

예제

DATETIME'1999-10-10'는 ‘1999/10/10 12:00:00.000’으로 저장된다.
DATETIME'10-10'는 ‘2009/10/10 12:00:00.000’으로 저장된다. (연도가 생략되면 현재 연도가 자동으로 지정됨.)
