EXTRACT 연산자

설명

EXTRACT 연산자는 날짜-시간 수식의 값 일부분을 추출하는데 사용된다.

구문

EXTRACT ( field FROM date-time_argument )

field :
YEAR
MONTH
DAY
HOUR
MINUTE
SECOND
MILLISECOND

date-time_argument :
expression

  • field : 날짜-시간 수식에서 추출할 값을 지정한다.
  • date-time argument : 일부분을 추출할 날짜-시간 수식을 선언한다. 날짜-시간 인자의 수식은 TIME, DATE, TIMESTAMP, DATETIME 타입 중 하나의 값이어야 하며 NULL로 지정될 수도 있다. 추출된 값은 INTEGER 타입이며, 날짜-시간 인자가 NULL인 경우에는 NULL 값이 반환된다.
예제

다음은 날짜-시간 수식에서 월만 추출하는 예제이다.

SELECT EXTRACT(MONTH FROM TIMESTAMP '07/31/1994 8:15:00 pm') FROM db_root;

결과값: 7

다음은 DATETIME에서 밀리초만 추출하는 예제이다.

SELECT EXTRACT(MILLISECOND FROM DATETIME '07/31/1994 8:15:00.031 pm') FROM db_root;

결과값: 31
