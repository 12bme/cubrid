BIGINT

설명

BIGINT 데이터 타입은 정확한 수치형 타입이다. 정밀도는 19이고 스케일은 0이다. BIGINT 타입은 8바이트(64비트) 정수를 저장하며, 지원하는 범위는 -9,223,372,036,854,775,808에서 9,223,372,036,854,775,807이다.

BIGINT

참고 사항
  • 만약, 소수점 아래 숫자가 있는 값을 BIGINT에 지정하면 소수점 뒤의 숫자는 모두 절삭된다.
  • 정밀도와 표현할 수 있는 범위를 기준으로는 다음과 같이 정렬할 수 있다.
    SMALLINT ⊂ INTEGER ⊂ BIGINT ⊂ NUMERIC
예제

INTEGER에 8934를 지정하면 8934가 저장됨.
INTEGER에 7823467를 지정하면 7823467이 저장됨.
INTEGER에 89.8를 지정하면 89가 저장됨(소수점 뒤의 수치는 모두 절삭됨).
INTEGER에 3458901122를 지정하면 오류가 발생함(표현 가능 범위를 초과하면 오류 발생).
