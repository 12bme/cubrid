UPPER 함수

설명

UPPER 함수는 모든 소문자 ASCII 또는 ISO 8859-1 라틴 문자를 대문자로 변환한 형태의 문자열을 반환한다.

구문

UPPER ( upper_operand )

upper_operand :
character string
NULL

  • upper_operand : 대문자로 변환할 문자열을 지정한다. 값이 NULL이면 결과는 NULL이 반환된다.  
예제

다음은 문자열 'Cubrid'를 모두 대문자로 변환하여 반환하는 예제이다. 

SELECT UPPER('Cubrid')
FROM db_root;

결과값: 'CUBRID'
