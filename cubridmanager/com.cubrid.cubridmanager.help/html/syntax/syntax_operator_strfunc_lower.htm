LOWER 함수

설명

LOWER 함수는 모든 대문자 ASCII 또는 ISO 8859-1 라틴 문자를 소문자로 변환한 형태의 문자열을 반환한다.

구문

LOWER (lower_operand )

lower_operand :
character string
NULL

  • lower_operand : 소문자로 변환할 문자열을 지정한다. 값이 NULL이면 결과는 NULL이 반환된다.
예제

다음은 문자열 'Cubrid'를 모두 소문자로 반환하는 예제이다.

SELECT LOWER('Cubrid')
FROM db_root;

결과값: 'cubrid'
