_db_auth

클래스에 대한 사용자 권한 정보를 나타내며, grantee에 인덱스가 생성되어 있다.

속성명

데이터 타입

설명

grantor

db_user

권한 부여자

grantee

db_user

권한 받은자

class_of

_db_class

권한부여 대상인 클래스 객체

auth_type

VARCHAR(7)

부여된 권한 타입 이름

is_grantable

INTEGER

권한 받은 클래스에 대해 다른 사용자에게 권한을 부여할 수 있으면 1, 아니면 0

CUBRID가 지원하는 권한 타입은 다음과 같다.

  • SELECT
  • INSERT
  • UPDATE
  • DELETE
  • ALTER
  • INDEX
  • EXECUTE
예제

다음 예제에서는 클래스 ‘glo’에 정의되어 있는 권한 정보를 검색한다.

csql> select grantor.name, grantee.name, auth_type
csql> from _db_auth
csql> where class_of.class_name = 'glo';
csql> ;xrun
