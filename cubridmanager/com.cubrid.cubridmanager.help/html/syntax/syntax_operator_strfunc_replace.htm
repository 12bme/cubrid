REPLACE 함수

설명

REPLACE 함수는 식 내의 문자열을 다른 문자열로 바꾼 후 결과를 반환한다.

구문

REPLACE( char1, search_string [, replacement_string ] )

char1 :
character string
NULL

search_string :
character string
NULL

replacement_string :
character string
NULL

REPLACE 함수는 search_string을 replacement_string으로 치환한 char1을 반환한다.

  • char1 : 원본 문자열을 지정한다. 값이 NULL이면 결과로 NULL이 반환된다.
  • search_string : 값을 바꿀 문자열을 지정한다. 값이 NULL이면 결과로 NULL이 반환된다.
  • replacement_string : search_string을 대체할 문자열을 지정한다. 값이 생략되면 char1에서 search_string을 제거하여 반환한다. 값이 NULL이면 결과로 NULL이 반환된다.

예제

다음은 gender 컬럼의 값이 M이면 Men's로 대체하여 반환하는 예제이다.

SELECT  REPLACE(gender, 'M', 'Men''s' ), sports, name FROM event
WHERE name = '100m Freestyle'
AND gender = 'M';

결과값: Men's Swimming 100m Freestyle
