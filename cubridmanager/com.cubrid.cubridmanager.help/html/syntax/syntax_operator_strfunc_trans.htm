TRANSLATE 함수

설명

TRANSLATE 함수는 문자열을 대응하는 다른 문자로 치환한다. 예를 들어, TRANSLATE(char1, 'abc', '123')은 char1의 모든 a를 1로, b를 2로, c를 3으로 치환한다.

구문

TRANSLATE( char1, from [, to] )

char1 :
character string
NULL

from :
character string
NULL

to :
character string
NULL

TRANSLATE 함수는 char1 문자열에서 from 문자열의 문자를 to 문자열의 대응되는 문자로 치환하여 반환한다.

  • char1 : 원본 문자열을 지정한다. 값이 NULL이면 결과로 NULL이 반환된다.
  • from : 치환할 대상 문자열을 지정한다. from 문자열의 수가 to 문자열의 수보다 크면, from 문자열에서 to 문자열에서 대응되지 않는 문자는 char1에서 삭제된다. 값이 NULL이면 결과로 NULL이 반환된다.
  • to : from 문자열에 대응할 문자열을 지정한다. 값이 NULL이면 결과로 NULL이 반환된다.
예제

다음은 1을 A로, 2를 B로, 3을 C로, 4를 D로, 5를 E로 치환하여 출력하는 예제이다.

SELECT TRANSLATE('1234567890123', '12345', 'ABCDE' ) FROM db_root;

결과값: ABCDE67890ABC
