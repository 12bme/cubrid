가상 테이블 삭제

설명

가상 테이블은 DROP 구문을 이용하여 삭제할 수 있다. 가상 테이블을 삭제하는 방법은 일반 테이블을 삭제하는 방법과 동일하다.

구문

DROP [ VIEW | VCLASS ] view_spec [ { ,class_spec }_ ] [ ; ]

view_spec :
single_view_spec
( single_view_spec [ { , single_view_spec }_ ] )

single_view_spec :
[ ONLY ] view_name
ALL view_name [ ( EXCEPT view_spec ) ]

  • view_name : 삭제하려는 가상 테이블의 이름을 지정한다.
예제

다음은 가상 테이블 swimming을 삭제하는 예제이다.

DROP VIEW swimming;
