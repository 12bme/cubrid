_db_index_key

인덱스에 대한 키 정보이며 index_of에 대한 인덱스가 생성되어 있다.

속성명

데이터 타입

설명

index_of

_db_index

키 속성이 속하는 인덱스

key_attr_name

VARCHAR(255)

키를 구성하는 속성명

key_order

INTEGER

키에서 속성이 위치한 순서로 0부터 시작함

asc_desc

INTEGER

속성 값의 순서가 내림차순이면 1, 그렇지 않으면 0
예제

다음 예제에서는 클래스에 속하는 인덱스명을 검색한다.

csql> select class_of.class_name, sequence(select key_attr_name
csql>                                      from _db_index_key k
csql>                                      where k in i.key_attrs)
csql> from _db_index i
csql> where key_count >= 2;
