TO_NUMBER 함수

설명

TO_NUMBER 함수는 문자열을 숫자 포맷에 맞는 NUMERIC 타입으로 변환한다.  

구문

TO_NUMBER(string_argument[, format_argument ])

string_argument :
character strings
NULL

format_argument :
character strings
NULL

반환값의 전체 자릿수와 소수점 이하 자릿수는 string_argument의 실제 숫자값의 전체 자릿수와 소수점 이하 자릿수로 지정된다. 예를 들어, '123.45'의 전체 자릿수는 5, 소수점 이하 자릿수는 2가 된다. '1.23e+1'의 전체 자릿수는 3, 소수점 이하 자릿수는 1이 된다.

  • string_argument : NUMERIC 타입으로 변환할 문자열을 지정한다. 값이 NULL이면 결과로 NULL이 반환된다.
  • format_argument : 반환할 값의 양식을 지정한다. 생략되면, NUMERIC 포맷으로 문자열을 반환한다. 값이 NULL이면 결과로 NULL이 반환된다.
예제
  • 예제 1
  • 다음은 문자열 '-1234'를 NUMERIC 타입으로 변환하여 반환하는 예제이다.

SELECT TO_NUMBER('-1234') FROM db_root;

  • 결과값: -1234
  • 예제 2
  • 다음은 문자열 '0.123'을 0.0000 과 같은 양식의 NUMERIC 타입으로 변환하여 반환하는 예제이다.

SELECT TO_NUMBER('0.123','0.0000') FROM db_root;

  • 결과값: .1230
