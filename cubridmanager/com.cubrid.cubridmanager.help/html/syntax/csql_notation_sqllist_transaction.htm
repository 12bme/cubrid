SET TRANSACTION ISOLATION LEVEL
<isolation level spec>

<isolation level spec>

<expression>
| SERIALIZABLE
| CURSOR STABILITY
| <class isolation> [ , <instance isolation> ]
| <instance isolation> [ , <class isolation> ]

<class isolation>

REPEATABLE READ { CLASS | SCHEMA }
| READ COMMITTED { CLASS | SCHEMA }
| READ UNCOMMITTED { CLASS | SCHEMA }

<instance isolation>

REPEATABLE READ INSTANCES
| READ COMMITTED INSTANCES
| READ UNCOMMITTED INSTANCES

<timeout spec>

INFINITE
| OFF
| <unsigned integer literal>
| <variable>>
