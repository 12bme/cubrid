loadjava 유틸리티

설명

컴파일된 Java 파일이나 JAR(Java Archive) 파일을 CUBRID로 로딩하기 위해서 loadjava 유틸리티를 사용한다. loadjava 유틸리티를 사용하여 Java *.class 파일이나 *.jar 파일을 로딩하면 해당 데이터베이스 경로에 해당 파일이 이동시킨다.

구문

loadjava <option> database-name java-class-file

  • database-name : Java 파일을 로드 하려고 하는 데이터베이스 이름
  • java-class-file : 로드 하려는 Java 클래스 파일 이름 또는 jar 파일 이름
  • < option> :
    • -y : 이름이 같은 클래스 파일이 존재하면 자동으로 덮어쓰기 한다. 디폴트는 no이다. 만약 – y 옵션을 명시하지 않고 로딩할 때 이름이 같은 클래스 파일이 존재하면 덮어쓰기를 할 것인지 묻는다.
