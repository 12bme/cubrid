GRANT/REVOKE 문

<authorization statement>

GRANT <privileges> ON <class specification comma list>
TO <user name comma list> [ WITH GRANT OPTION ]
REVOKE <privileges> ON <class specification comma list>
FROM <user name comma list>

<privileges>

ALL [ PRIVILEGES ]
| <privilege comma list>

<privilege>

SELECT
| INSERT
| DELETE
| UPDATE
| INDEX
| ALTER
| EXECUTE

예제

GRANT SELECT, INSERT, UPDATE ON employee TO jones;

GRANT ALL PRIVILEGES ON person, student, TO smith, brown;

REVOKE INSERT, UPDATE ON employee FROM smith;

REVOKE SELECT ON manufacturing_site FROM jones, brown;
