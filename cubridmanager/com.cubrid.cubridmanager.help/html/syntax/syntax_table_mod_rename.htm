테이블 이름 변경

설명

테이블 이름은 RENAME 구문을 사용하여 변경할 수 있다.

구문

RENAME  [ TABLE | CLASS | VIEW | VCLASS] old_talbe_name AS new_table_name [ ; ]

  • old_table_name : 변경할 테이블의 이름을 지정한다.
  • new_table_name : 테이블의 새로운 이름을 지정한다.
예제

다음은 stadium 테이블의 이름을 stadium_info로 변경하는 예제이다.

RENAME TABLE stadium AS stadium_info;

주의 사항

테이블의 이름 변경은 테이블의 소유자, DBA, DBA의 멤버만이 할 수 있으며, 이 외의 사용자는 소유자나 DBA로부터 이름을 변경할 수 있는 권한을 받아야 한다(권한 관련 사항은 "권한 부여" 참조).
