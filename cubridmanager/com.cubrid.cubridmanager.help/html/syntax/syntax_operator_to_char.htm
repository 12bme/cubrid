날짜 타입(DATE, TIME, TIMESTAMP, DATETIME)
설명

TO_CHAR 함수는 날짜-시간 타입(DATE, TIME, TIMESTAMP, DATETIME)을 "날짜-시간" 포맷에 맞는 VARCHAR 타입으로 변환한다. 날짜-시간 포맷이 생략되었을 때에 DATE 타입이면 'MM/DD/YYYY' 포맷이, TIME 타입이면 'HH:MI:SS' 포맷이, TIMESTAMP 타입이면 'HH:MI:SS AM MM/DD/YYYY' 포맷이, DATETIME 타입이면 'HH:MI:SS.FF AM MM/DD/YYYY' 포맷이 적용된다.

구문

TO_CHAR( date_time_argument [, format_argument[, date_lang_string_literal ]] )

date_time_argument :
date
time
timestamp
datetime
NULL

format_argument :
character strings (날짜 포맷 표 참조)
NULL

date_lang_string_literal : (date_lang_string_literal 표 참조)
'en_US'
'ko_KR'

  • date_time_argument : VARCHAR 타입으로 변경할 날짜-시간 타입 값을 지정한다. 값이 NULL인 경우에는 NULL이 반환된다.
  • format_argument : 반환할 값의 양식을 지정한다. 값이 NULL인 경우에는 NULL이 반환된다.
  • date_lang_string_literal : 반환값에 적용할 언어를 지정한다(date_lang_string_literal 표 참조). 디폴트 값은 'en_US'이다. CUBRID_DATE_LANG 환경 변수를 지정하여 수정할 수 있다.

예제
  • 예제 1
  • 다음은 DATE 타입에서 날짜만 반환하는 예제이다.

SELECT TO_CHAR(DATE '12/25/1999', 'dd') FROM db_root;

  • 결과값: 25
  • 예제 2
  • 다음은 TIME 타입에서 시만 반환하는 예제이다.

SELECT TO_CHAR(TIME '10:20:30 AM', 'HH24') FROM db_root;

  • 결과값: 10
  • 예제 3
  • 다음은 TIMESTAMP 타입에서 월 이름, 축약된 월 이름, 연도를 반환하는 예제이다.

SELECT TO_CHAR(TIMESTAMP '10:20:30 AM 12/25/1999', 'MONTH MON yyyy') FROM db_root;

  • 결과값: DECEMBER DEC 1999
  • 예제 4
  • 다음은 TIMESTAMP에서 요일을 숫자로 반환하는 예제이다.

SELECT TO_CHAR(TIMESTAMP '10:20:30 AM 12/25/1999', 'D') FROM db_root;

  • 결과값: 7
  • 예제 5
  • 다음은 DATETIME에서 초와 밀리초를 반환하는 예제이다.

SELECT TO_CHAR(DATETIME'10:12:13 6/5/2009','ss ff') FROM db_root;

  • 결과값: 13 000
숫자 타입(NUMERIC)
설명

TO_CHAR 함수는 NUMERIC 타입을 숫자 포맷에 맞는 VARCHAR 타입으로 변환한다.

구문

TO_CHAR(numeric_argument[, format_argument ])

numeric_argument :
numeric(decimal)
integer
smallint
bigint
float(real)
double
NULL

format_argument :
character strings (숫자 포맷 표 참조)
NULL

  • numeric_argument : VARCHAR 타입으로 변환할 숫자값을 지정한다. 값이 NULL이면 결과로 NULL이 반환된다.

  • format_argument : 반환할 값의 양식을 지정한다. 생략되면, 숫자 타입값에 따라서 기본 포맷을 따른다. 값이 NULL이면 결과로 NULL이 반환된다.

포맷 구성 요소

"9"의 개수가 반환될 유효숫자 자릿수를 나타낸다. 숫자 앞쪽의 0에 대해서는 공백이 반환된다.

숫자 앞쪽의 0에 대해서 공백이 아니라 0을 반환한다.

지정된 위치에 양수에 대해서는 "+", 음수에 대해서는 "-"를 반환한다.

C

C9999

지정된 위치에 ISO 통화 기호를 반환한다.

,

9,999

지정된 위치에 쉼표(",")를 반환한다.

.

9.999

지정된 위치에 숫자의 정수 부분과 소수 부분을 구분하는 마침표(".")를 반환한다.

EEEE

9.99EEEE

과학적 기수법(scientific notation)을 반환한다.
예제
  • 예제 1
  • 다음은 숫자를 사인 부호와 4자리의 VARCHAR 타입으로 반환하는 예제이다.

SELECT TO_CHAR(1234,'S9999') FROM db_root;

  • 결과값: +1234
  • 예제 2
  • 다음은 과학적 기수법으로 표기된 숫자를 VARCHAR 타입으로 반환하는 예제이다.

SELECT TO_CHAR(1.234E-4) FROM db_root;

  • 결과값: 0.0001234
