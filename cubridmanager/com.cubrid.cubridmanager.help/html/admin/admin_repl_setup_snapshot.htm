스냅샷 동기화

스냅샷 동기화(repl_make_snapshot)는 마스터 데이터베이스의 복제 대상 클래스 중 복제에 참여하지 않던 것을 운영 중에 복제에 추가하고자 할 때 사용하는 유틸리티이다. 적용 대상은 repl_make_group 유틸리티로 마스터 데이터베이스의 일부 클래스만 복제하는 슬레이브 데이터베이스이다.

구문

repl_make_snapshot -m masterdb_name -s slavedb_name -d distdb_name -cf classes_file_name [-p dist_db_password]

  • –m masterdb_name : 마스터 데이터베이스 이름
  • –s slavedb_name : 슬레이브 데이터베이스 이름
  • –d distdb_name : 배포 데이터베이스 이름
  • –cf classes_file_name : 추가할 클래스 목록에 대한 파일 이름으로, 클래스는 공백 또는 쉼표(,)로 구분
  • –p dist_db_password : 배포 데이터베이스에 대한 DBA 암호
