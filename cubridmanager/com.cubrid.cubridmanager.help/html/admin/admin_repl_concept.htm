개요

데이터베이스 복제는 하나의 데이터베이스에 저장된 객체를 물리적으로 분리된 다른 데이터베이스에 복사하여 두 개 이상의 데이터베이스 서버에서 사용될 수 있도록 하는 분산 데이터베이스 기술 중 하나이다. 복제 기술은 다음과 같은 용도로 사용할 수 있다.

  • 같은 객체를 이용하는 응용의 접근을 여러 데이터베이스 서버에 분산시킴으로써 성능을 높일 수 있다.
  • 복제된 데이터베이스 서버를 다른 용도로 사용할 수 있도록 하여 서로 다른 운영 요구 사항을 만족시킬 수 있다.
  • 운영 중인 데이터베이스 서버에 장애가 발생했을 경우 복제 데이터베이스 서버로 대체함으로써 장애 발생 시 긴급하게 대처할 수 있다.

참고 현재(2009년 2월 기준) 복제 기능은 UNIX 계열에서만 지원하고 Windows에서는 지원하지 않는다.
