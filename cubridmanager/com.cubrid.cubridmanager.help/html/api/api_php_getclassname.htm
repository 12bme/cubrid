cubrid_get_class_name

설명

cubrid_get_class_name 함수는 OID로부터 클래스 이름을 얻는 데 사용된다.

구문

mixed  cubrid_is_instance (int conn_handle, string oid)

  • conn_handle : 연결 핸들
  • oid : 존재 여부를 알기 원하는 인스턴스의 OID
리턴 값
  • 성공 : 클래스 이름
  • 실패 : false
예제

$target_oid = cubrid_get ($con, $oid, "customer");
$class_name = cubrid_get_class_name ($con, $target_oid);
if ($class_name) {
  echo "class name of $oid is $class_name\n";
} else {
  echo "error\n";
}
