cubrid_close_request

설명

cubrid_close_request 함수는 req_handle 인수에 주어진 요청 핸들을 닫고, 핸들에 관련된 메모리 영역을 해제한다.

구문

int cubrid_close_request (int req_handle)

  • req_handle : 요청 핸들
리턴 값
  • 성공 : true
  • 실패 : false

$con = cubrid_connect ("dbsvr.cubrid.com", 12345, "demodb");
