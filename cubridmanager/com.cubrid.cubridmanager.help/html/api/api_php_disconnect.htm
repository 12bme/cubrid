cubrid_disconnect

설명

cubrid_disconnect 함수는 현재 진행중인 트랜잭션을 중지하고, 서버와의 연결을 해제 및 연결 핸들을 닫는다. 만약 현재까지 아직 닫히지 않은 요청 핸들이 있으면 모두 닫히게 된다.

구문

int cubrid_disconnect (int conn_handle)

  • conn_handle : 연결 핸들
리턴 값
  • 성공 : true
  • 실패 : false
예제

$con = cubrid_connect ("210.211.133.100", 12345, "demodb");
if ($con) {
   echo "connected successfully";
   $req = cubrid_execute ( $con, "insert into person values(1,'James')");
   if ($req) {
      cubrid_close_request ($req);
      cubrid_commit ($con);
   } else {
      cubrid_rollback ($con);
   }
   cubrid_disconnect ($con);
}
