연결 문자열(connection string) 구성

ADO(ActiveX Data Object) 혹은 ADO.net을 이용하여 CUBRID OLE DB Provider 프로그래밍을 할 때 연결 문자열(connection string)은 다음과 같이 작성한다.

항목

예

설명

Provider

CUBRIDProvider

공급자 이름

Data Source

demodb

데이터베이스 이름

Location

192.168.1.11

서버 IP

User ID

PUBLIC

사용자 ID

Password

xxx

비밀번호

Port

30000

브로커 Port 번호

Fetch Size

100

Fetch 크기

위의 예를 이용한 연결 문자열은 다음과 같다.

 

"Provider = CUBRIDProvider;Data Source = demodb;Location = 192.168.1.11;User ID = PUBLIC;Password =xxx;Port = 30000;Fetch Size = 100"
