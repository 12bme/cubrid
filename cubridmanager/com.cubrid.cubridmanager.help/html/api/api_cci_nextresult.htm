cci_next_result

설명

cci_execute 시 CCI_EXEC_QUERY_ALL flag가 설정된 경우 실행된 다음 질의의 결과를 가져온다. next_result에 의해 얻은 질의에 대한 정보는 cci_get_result_info를 통해서 얻을 수 있다. next_result가 성공적으로 수행될 경우 현재의 질의에 대한 정보로 수정된다.

에러 코드가 CAS_ER_NO_MORE_RESULT_SET일 경우 더 이상의 result set이 존재하지 않는다.

구문

int cci_next_result(int req_handle, T_CCI_ERROR *err_buf)

  • req_handle : (IN) prepare된 SQL 문의 요청 핸들
  • err_buf : (OUT) 데이터베이스 에러 버퍼
리턴 값
  • 성공 :
    • SELECT (sync mode) : 결과 개수, (async mode) : 0
    • INSERT, UPDATE : 반영된 튜플 개수
    • 기타 : 0
  • 실패 : 에러 코드
