cubrid_seq_drop

설명

cubrid_seq_drop 함수는 주어진 SEQUENCE 타입의 속성에 원하는 원소를 데이터베이스에서 삭제한다.

구문

int cubrid_seq_drop(int conn_handle, string oid, string attr_name, int index)

  • conn_handle : 연결 핸들
  • oid : 원하는 인스턴스의 OID
  • attr_name : 인스턴스의 원하는 속성이름
  • index : 삭제하고 싶은 원소의 인덱스. 디폴트 값은 1.
리턴 값
  • 성공 : true
  • 실패 : false
예제

$elems = cubrid_col_get ($con, $oid, "style");
$i = 1;
while (list ($key, $val) = each($elems)) {
if ($val == "1") {
    echo $val;
    cubrid_seq_drop ($con, $oid, "style", $i);
  }
  $i++;
}
