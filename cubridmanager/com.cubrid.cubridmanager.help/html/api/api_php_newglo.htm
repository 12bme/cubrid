cubrid_new_glo

설명

cubrid_new_glo 함수는 요청된 클래스(glo 클래스)에 glo 인스턴스를 생성한다. 생성하는 glo는 LO 타입이며, file_name 파일에 저장한다.

구문

mixed cubrid_new_glo (int conn_handle, string class_name, string file_name)

  • conn_handle : 연결 핸들
  • class_name : glo를 생성하려고 하는 클래스의 이름
  • file_name : glo를 생성한 후 glo에 저장하고 싶은 파일의 이름
리턴 값
  • 성공 : 생성된 인스턴스의 OID
  • 실패 : false
예제

$oid = cubrid_new_glo ($con, "glo", "input.jpg");
if ($oid){
   $req = cubrid_execute ($con,
                          "insert into person(image) values ($oid)");
   if ($req) {
      echo "image inserted successfully";
      cubrid_close_request ($req);
      cubrid_commit($con);
   }
}
