cci_get_result_info

설명

prepare된 SQL 문이 SELECT일 경우, 컬럼 정보에 대한 포인터를 가져온다. SELECT에 대한 핸들이 아닐 경우 NULL을 반환하고 num 값은 0이다. 이 때, CCI_GET_RESULT_INFO_TYPE() 등을 호출해서는 안 된다.

구문

T_CCI_COL_INFO* cci_get_result_info(int req_handle, T_CCI_SQLX_CMD *cmd_type, int *num)

  • req_handle : (IN) prepare된 SQL 문에 대한 요청 핸들
  • cmd_type : (OUT) command 타입
  • num : (OUT) SELECT 문의 컬럼 개수(cmd_type이 SQLX_CMD_SELECT일 경우)
리턴 값
  • 성공 : result info 포인터
  • 실패 : NULL
