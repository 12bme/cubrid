cubrid_affected_rows

설명

cubrid_affected_rows 함수는 SQL 문(INSERT, DELETE, UPDATE)에 의해 적용 받은 행의 개수를 얻는데 사용된다.

구문

int cubrid_affected_rows (int req_handle)

  • req_handle : 요청 핸들
리턴 값
  • 성공 : SQL 문에 의해 적용 받은 행의 개수를 반환
  • 실패 : -1
예제

$req = cubrid_execute ($con, "delete from person where name like 'j%' ");
if ($req) {
   $row_count = cubrid_affected_rows ($req);
   echo $row_count;
   cubrid_close_request ($req);
}
