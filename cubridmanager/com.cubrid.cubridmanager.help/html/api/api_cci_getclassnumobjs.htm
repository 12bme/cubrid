cci_get_class_num_objs

설명

class_name 클래스의 객체 개수와 사용하고 있는 페이지 수를 가져온다. flag가 1일 경우 대략의 값을 가져오고, 0일 경우 정확한 값을 가져온다.

구문

intcci_get_class_num_objs(int conn_handle, char *class_name, int flag, int *num_objs, int *num_pages, T_CCI_ERROR *err_buf)

  • conn_handle : (IN) 연결 핸들
  • class_name : (IN) 클래스 이름
  • flag : (IN) 0 또는 1
  • num_objs : (OUT) 객체 수
  • num_pages : (OUT) 페이지 수
  • err_buf  : (OUT) 데이터베이스 에러 버퍼
리턴 값
  • 에러 코드 (0 : 에러 없음)
