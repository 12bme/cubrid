cubrid_commit

설명

cubrid_commit 함수는 conn_handle이 가리키는 접속에서 현재 진행중인 트랜잭션을 커밋한다. cubrid_commit 함수가 호출된 이후에는 서버와의 연결이 끊어지지만 연결 핸들은 유효하다.

구문

int cubrid_commit (int conn_handle)

  • conn_handle : 연결 핸들
리턴 값
  • 성공 : true
  • 실패 : false
예제

$req = cubrid_execute ($oid, "insert into person values
    (2,'John')");
if ($req) {
   cubrid_close_request ($req);
