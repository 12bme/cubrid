cci_col_size

설명

set(seq) 속성의 개수를 가져온다.

구문

intcci_col_size (int conn_handle, char *oid_str, char *col_attr, int *col_size, T_CCI_ERROR *err_buf)

  • conn_handle : (IN) 연결 핸들
  • oid_str  : (IN) oid
  • col_attr : (IN) collection 속성 이름
  • col_size : (OUT) collection 크기 (-1 : NULL)
  • err_buf  : (OUT) 데이터베이스 에러 버퍼
리턴 값
  • 에러 코드 (0 : 에러 없음)
