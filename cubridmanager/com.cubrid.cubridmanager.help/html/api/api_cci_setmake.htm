cci_set_make

설명

새로운 CCI_A_TYPE_SET type의 set을 만든다. 만들어진 set은 cci_bind_param()을 통해 CCI_A_TYPE_SET으로 서버에 전달된다. cci_set_make()에 의해 만들어진 set은 반드시 cci_set_free()를 통해 사용된 메모리를 제거해야 한다. u_type에 대한 value의 타입은 다음과 같다.
