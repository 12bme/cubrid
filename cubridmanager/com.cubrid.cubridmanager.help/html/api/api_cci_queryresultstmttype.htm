CCI_QUERY_RESULT_STMT_TYPE

설명

cci_execute_batch 질의에 의한 statement type을 가져오는 매크로이다. 파라미터의 query_result가 NULL인지, index가 유효한지에 대한 검사는 하지 않는다.

구문

#define CCI_QUERY_RESULT_STMT_TYPE(T_CCI_QUERY_RESULT* query_result, int index)

  • query_result : (IN) cci_execute_batch에 의한 쿼리 결과
  • index : (IN) 컬럼 인덱스(base : 1)
리턴 값
  • statement type (T_CCI_SQLX_CMD)
