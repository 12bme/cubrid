cci_cursor

설명

요청 핸들에 설정된 커서를 이동시킨다. cci_cursor(req_handle, 1, CCI_CURSOR_FIRST, err_buf)는 항상 첫 번째 위치로 커서를 이동시키고, cci_cursor(req_handle, 1, CCI_CURSOR_LAST, err_buf)는 마지막 위치로 커서를 이동시킨다. cci_cursor(req_handle, 1, CCI_CURSOR_CURRENT, err_buf) 는 커서의 현재 위치에서 다음 위치로 이동시킨다. 유효한 커서 위치가 아닐 경우 CCI_ER_NO_MORE_DATA를 반환한다.

구문

int cci_cursor(int req_handle, int offset, T_CCI_CURSOR_POS origin, T_CCI_ERROR *err_buf)

  • req_handle : (IN) 요청 핸들
  • offset : (IN) 이동할 오프셋
  • origin : (IN) 커서 위치
  • err_buf : (OUT) 데이터베이스 에러 버퍼
리턴 값
  • 에러 코드 (0 : 에러 없음)
