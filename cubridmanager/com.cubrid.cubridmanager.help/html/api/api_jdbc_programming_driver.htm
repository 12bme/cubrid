CUBRID JDBC 드라이버

CUBRID JDBC 드라이버는 Java로 작성된 응용 프로그램에서 CUBRID 데이터베이스를 접속할 수 있도록 해준다. CUBRID JDBC 드라이버 v1.0은 JDBC 1.2 스펙을 기준으로 개발되었다. CUBRID JDBC 드라이버는 CUBRID CAS를 통하여 DBMS와 연결하는 Pure Java Driver for Database Middleware (type 3)에 해당한다. CUBRID JDBC 드라이버는 JDK 1.5에서 컴파일한 것을 기본으로 제공하며,  JDK 1.4나 JDK 1.6을 사용하는 경우에는 릴리스 노트나 JDBC 환경 설정 부분을 참고하기 바란다.

CUBRID JDBC 드라이버 등록

JDBC 드라이버 등록은 Class.forName(driver-class-name) 명령을 사용하며, 아래는 CUBRID JDBC 드라이버를 등록하기 위해 cubrid.jdbc.driver.CUBRIDDriver 클래스를 로드하는 예제이다.

import java.sql.*;
import cubrid.jdbc.driver.*;
 
public class LoadDriver {
   public static void main(String[] Args) {
       try {
           Class.forName("cubrid.jdbc.driver.CUBRIDDriver");
       } catch (Exception e) {
           System.err.println("Unable to load driver.");
           e.printStackTrace();
       }
