cci_glo_new

설명

class_name의 glo 클래스에 filename에 의한 파일로 glo 객체를 만든다. filename이 NULL인 경우 glo 객체만을 생성한다(cci 2.3 이상).

구문

intcci_glo_new(int conn_handle, char *class_name, char *filename, char *oid_str, T_CCI_ERROR *err_buf)

  • conn_handle : (IN) 연결 핸들
  • class_name : (IN) glo 클래스 이름
  • filename : (IN) 소스 파일
  • oid_str : (OUT) glo 객체(OID 스트링)
  • err_buf  : (OUT) 데이터베이스 에러 버퍼
리턴 값
  • 에러 코드 (0 : 에러 없음)
